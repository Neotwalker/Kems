# InkFlow Leads — лендинг для продвижения тату-салонов

Готовая статическая версия сайта для загрузки на GitHub Pages. Дизайн сделан в тёмной неоновой стилистике, структура похожа на референс: hero, услуги, схема работы, путь клиента, кейсы, пакеты, отзывы, FAQ и форма заявки.

## Как загрузить на GitHub Pages

1. Создайте новый репозиторий на GitHub.
2. Загрузите все файлы из архива в корень репозитория.
3. Откройте Settings → Pages.
4. В Source выберите Deploy from a branch.
5. Выберите ветку `main` и папку `/root`.
6. Сохраните настройки и дождитесь публикации.

## Что заменить перед публикацией

- Название `InkFlow Leads` на название компании.
- Телефон `+7 000 000-00-00`.
- Почту `hello@example.com`.
- Ссылки на Telegram, политику конфиденциальности и реальные соцсети.
- Цены в блоке пакетов.
- Демонстрационные отзывы и сценарии кейсов на реальные.
- URL в JSON-LD внутри `index.html`: `https://example.com/`.

## Как подключить форму

Сейчас форма не отправляет данные на сервер — она показывает сообщение-заглушку. Для реального сайта можно подключить:

- Telegram Bot API через серверный обработчик;
- Formspree / Getform / Netlify Forms;
- WordPress + Contact Form 7;
- собственный PHP/Node.js обработчик.

В файле `assets/js/main.js` найдите обработчик `form.addEventListener('submit', ...)` и замените заглушку на отправку данных.
